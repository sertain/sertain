Command(timeout: Long = 0, unit: TimeUnit = TimeUnit.MILLISECONDS)

A command to run on the robot. The command may require a subsystem in order to ensure that only one command is running for a given subsystem at a time.

Parameters

timeout - the maximum amount of time this command should be run for

unit - the unit of time for timeout
